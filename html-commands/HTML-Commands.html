HTML Commands:

<img>
  • purpose: embeds am image into the document.
  • example: <img src="somefunnypicture.jpg" alt="some funny pictures">
  • attributes:
    • Attribute Values Meaning
    src URL source file
    height number, percentage the intrinsic height of the image, in pixels. Must be an interger without a unit.
    width number, percentage The intrinsic width of the image in pixels. Must be an interger without a unit.
    alt text Defines an alternative text description of the image.

<a>, </a>
  • purpose: anchor element, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address.
  • example:<a href="https://en.wikipedoa.org/wiki/cat">Wiki Cats.</a>
  • attributes:
    • Attribute Values Meaning
    href URL The URL that the hyperlink points to.
    target name_of_frame Where tp display the linked URL, as the name for a browsing context (a tab, window, or <iframe>).
    _blank usually a new tab, but users can configure browsers to open a new window instead.
    _parent the parent browsing context of the current one. If no partent, behaves as _self.
    _top the topmost browsing context (the "highest" context thata an ancestor of the current one), If no ancestors, behaves as _self.
    _self the current browsing context.(Default)
